# My Gallery

## Overview
A simple static web gallery with a landing page featuring navigation to Photos, Videos, and Upload sections.

## Project Structure
- `index.html` - Main gallery landing page
- `background.jpg` - Background image
- `Photos.gif`, `Videos.gif`, `Upload.gif` - Navigation button images
- `server.py` - Simple Python HTTP server for development

## Running Locally
The project runs on port 5000 using a Python HTTP server with cache control headers disabled.

## Deployment
Configured as a static site deployment serving files from the root directory.
